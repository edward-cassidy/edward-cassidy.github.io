Finishing up the Feedback Bombe

So far, we have only used the cycle from the menu, but it is also useful to add in all the connections from the menu, not just the ones in the cycle.

In our earlier example which only added four scrambler assemblies to the bombe, it meant that the θ cable was never connected to any other cable, so never had a chance to be connected.

However if we add the missing connection to θ, the Bombe can actually help rule out values of θ, as well as the variable we are testing, α.
