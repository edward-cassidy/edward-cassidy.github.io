Operation of the Bombe

Therefore, in the real Bombe it was designed as follows.

The operator would set it up as we have shown, using the menu to find a cycle, and connecting the relevant scramblers to the cables.

The operator would then somewhat arbitrarily choose stecker variable to be the test register (for example α), and a value for that variable to take (for example b).

The machine would then send a current through the chosen wire of the test register, and if all the wires in the test register become live (case 3), it rules out this initial scrambler position as being possible, and it automatically continues to try the next scrambler position.

The Bombe only stops when less not all of the wires in the test register are activated, which is why the choice of input wire are largely arbitrary (as in this case, making any of the wires live makes all the other wires live).

When the Bombe stops, an operator can then examine which wires are live in the test register, and proceed to manually determine whether this was a false stop or not.
