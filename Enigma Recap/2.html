Introduction

We will now recap on how exactly an Enigma machine encodes a message, as this knowledge will be vital over the following chapters. We will not consider every intricacy of how the machine is set up and operates, but we will provide a general overview which will suffice for understanding the workings of the Bombe.

An Enigma machine is visually similar to an old-fashioned typewriter, except instead of writing onto paper, there are 26 bulbs labelled with each letter of the alphabet. When a key is pressed on the machine, the bulb which lights up indicates the output of the encryption.
