The Scramblers

Now we need to make some connections between cables in order to draw conclusions about the values of stecker variables (thus draw contradictions).

We will start by considering how α is scrambled to β at RSP 5, as shown below:

Remember when drawing conclusions by hand, we assumed some initial scrambler positions as well as a value for α, specifically, we assumed α = b. We then used the fact that α is scrambled to β at RSP 5 to manually calculate the value of β.

In order to make the machine perform this for us, let's wire in a scrambler assembly (all three scramblers, and reflector) between the the αin and β cables.

We can then set that scrambler assembly to the assumed initial scrambler position incremented by 5, and then power on the b wire in the αin cable.

The wire in the β cable which becomes live is the concluded value for β!
