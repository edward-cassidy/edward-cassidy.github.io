Basics of the Bombe

The Bombe used a technique called a probable phrase attack.

This is where the code breakers guess that a portion of the ciphertext was encoded from some particular plaintext, which is what we will call a crib. If their guess is correct, they can then use that to break the rest of the message.

So if the code breakers were to intercept a long portion of ciphertext which contained the characters egffbhg, they might guess that this was originally babbage.

But is it ever possible to make accurate guesses?
