Analysis of New Modifaction

Due to this new feedback, we can gauge a lot more information about the accuracy of our guess based on the number of live wires.

Say we power on the b wire in the α cable, as we have done previously. Here is what we can deduce, depending on the number of live wires in the test register:

    Case 1:
  • Only the b wire is live. This means that there was no contradiction, and it is possible that we correctly guessed the initial scrambler position and value of α.

    • Case 2:
  • More than one wire is live, but not all of them. This is a contradiction, since α cannot take multiple values. So for the assumed initial scrambler positions, none of the values for α indicated by live wires are valid.
    However, some of the remaining values for α may still be valid. In particular, if there is exactly one wire in the α cable which is not live, then the letter represented by this wire is definitely valid value for α for the assumed scrambler position (this is essentially the complement of case 1).

    • Case 3:
  • All wires become live. This is a contradiction, and there is no valid value for α for the assumed initial scrambler positions.


Case 3 is the most important case for the operation of the Bombe, as we can completely rule out the assumed initial scrambler position if this case occurs.
