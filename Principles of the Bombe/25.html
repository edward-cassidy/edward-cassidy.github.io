The concepts in this chapter form the basis of how we work out how the Enigma machine was initially set up, before the crib was encoded. Once we have worked this out, we can then decrypt the rest of the ciphertext using these settings.

Currently, we have to assume every possible initial scrambler position in turn, and for each, determine which steckerings for a given letter in the crib are valid.

Supposing we chose to test a, we would have to set α = a, then α = b, then α = c, ... and so on until α = h. We don't need to try any letters other than a, since if we can't find any valid steckering for a, then there cannot be any way to set up the plug board given the assumed initial scrambler position.

Hopefully, we will find that most pairs of assumptions lead to a contradiction, so the remaining few can be tested by hand. However, how many contradictions we find depends on factors such as the number of cycles in the menu, so there is still somewhat of a factor of chance that our crib produces a strong menu.
